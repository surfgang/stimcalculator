# Stim Calculator

Estimate ADHD medication onset, peak, and wear-off times from your dose time and dosage (Vyvanse, Ritalin IR/XR, dextroamphetamine) for when you can't take your dose in the morning.

Not medical advice. Always consult a professional before starting medications.

**Live demo:** https://YOUR_USERNAME.github.io/stimcalculator/ (after GitHub Pages is enabled)

## Run locally

```bash
cd stimcalculator
python3 -m http.server 8080
```

Open http://localhost:8080 (ES modules need a local server, not `file://`).
